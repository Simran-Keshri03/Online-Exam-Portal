# Online Exam Portal
A simple Online Exam Portal developed using HTML, CSS, and JavaScript. Users can attempt multiple-choice questions, navigate between them, and receive their final score after completing the exam.

Features

✅ Start Exam Screen

✅ Multiple Choice Questions

✅ Previous / Next Navigation

✅ Option Selection Highlight

✅ Prevents moving ahead without selecting an answer

✅ Automatic Score Calculation

✅ Final Result Display
